# 🍳 Cooking Bot - Телеграм бот с рецептами

Бот для просмотра блюд, добавления в избранное и управления рецептами.

## 📋 Функционал

### Для пользователей:
- 🍽️ **Просмотр случайных блюд** - каждый день новые рецепты
- ⭐ **Добавление в избранное** - сохраняйте понравившиеся рецепты
- 📋 **Просмотр избранного** - список всех сохранённых блюд
- 💎 **Система лимитов** - 3 бесплатных просмотра

### Для администраторов:
- ➕ Добавление новых блюд
- 🗑️ Удаление блюд

## 🚀 Установка и запуск

**python 3.10+ должен быть уже установлен на вашем компьтере. Для запуска вам необходимо:** 

1. **Скачайте репозиторий**
2. **Установите зависимости командой `pip install -r requirements.txt`**
3. **Так же вам необходимо получить токен от вашего бота:**
- **Переходим @BotFather к Главному боту всех ботов телеграмма и получаем ключик**
- **Создаем файл .env в корне проекта и заполняем ключ в следующем формате: `TELEGRAM_TOKEN='ваш токен'`**
4. **запускаем бота командой 'python bot.py'**
